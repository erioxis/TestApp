# README

Базу данных настраивать в /config/database.yml
По стандарту:
database: testapp
username: testuser
password: 312004

Для запуска проекта:
rails server
В браузере заходим на http://localhost:3000/
Заполняем форму и нажимаем отправить.
Формы можно просмотреть через localhost:3000/cargo_requests/номерформы
